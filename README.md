# TaskHub

TaskHub – це простий веб-додаток для управління завданнями.  
Реалізовано на Django.

## Можливості
- створення, редагування та видалення завдань
- додавання робітників
- використання тегів та проєктів
- авторизація користувачів

## Використані технології
- Python 3.11
- Django 5.x
- SQLite (за замовчуванням)

## ⚙️ Встановлення та запуск

1. Клонувати репозиторій:
   git clone https://github.com/your-username/taskhub.git
   
#### python -m venv venv
#### source venv/bin/activate   
#### pip install -r requirements.txt
#### python manage.py migrate
#### python manage.py createsuperuser
#### python manage.py runserver
